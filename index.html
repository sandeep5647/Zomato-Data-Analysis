Zomato Data Analysis¶

This Jupyter Notebook contains data analysis and visualizations for a Zomato dataset. The analysis answers the following questions:

  1. What type of restaurant do the majority of customers order from?
  2. How many votes has each type of restaurant received?
  3. What are the ratings that the majority of restaurants have received?
  4. What is the average spending of couples per order?
  5. Which mode (online or offline) has received the maximum rating?
  6. Which type of restaurant received more offline orders?

# Convert all values in the column to strings, then remove commas, and convert to numeric
data['approx_cost(for two people)'] = (
    data['approx_cost(for two people)']
    .astype(str)  # Convert to string
    .str.replace(',', '')  # Remove commas
    .replace('nan', None)  # Handle cases where values are 'nan' (as strings)
    .astype(float)  # Convert to numeric
)

fig, axes = plt.subplots(3, 2, figsize=(16, 16))

axes[0, 0].set_title('Restaurant Type Distribution', fontsize=14)

votes_by_type = data.groupby('listed_in(type)')['votes'].sum().sort_values(ascending=False)
axes[0, 1].barh(votes_by_type.index, votes_by_type.values, color='coral', alpha=0.7)
axes[0, 1].set_title('Total Votes by Restaurant Type', fontsize=14)
axes[0, 1].set_xlabel('Total Votes', fontsize=12)
axes[0, 1].set_ylabel('Restaurant Type', fontsize=12)

sns.boxplot(x='rate', data=data, ax=axes[1, 0], color='lightgreen')
axes[1, 0].set_title('Distribution of Restaurant Ratings', fontsize=14)
axes[1, 0].set_xlabel('Rating', fontsize=12)

average_spending = data['approx_cost(for two people)'].mean()
